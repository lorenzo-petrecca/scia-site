<h1 id="intro" class="docs-title scia-typo-uppercase scia-typo-normal scia-typo-black">Forms</h1>

<p class="intro">
Scia CSS provides a small set of form primitives to style common controls consistently:
inputs, textareas, selects, inline code, code blocks, and a simple switch toggle.
</p>

<br/>
---





<h2 id="field-wrapper" class="scia-h2 docs-subtitle">Field wrapper</h2>

Use <code class="scia-code">.scia-field</code> to group label, input, and helper text with consistent spacing.

<div className="playground-box">
```html
<div class="scia-field">
  <label class="scia-field-label" for="email">Email</label>
  <input class="scia-input" id="email" type="email" placeholder="name@example.com" />
  <div class="scia-field-help">We’ll never share your email.</div>
</div>
```
<div className='playground-box-content'>
<div className='graphic-example-title'>Field wrapper example</div>
<div className="scia-field">
  <label className="scia-field-label" htmlFor="demo-email">Email</label>
  <input
    id="demo-email"
    className="scia-input"
    type="email"
    placeholder="name@example.com"
  />
  <div className="scia-field-help">We’ll never share your email.</div>
</div>
</div>
</div>

<div class="scia-blockquote-info">
<p class="blockquote-paragraph"><code class="scia-code">.scia-field</code> is a flex column wrapper with a small gap and bottom spacing, so forms remain readable without custom CSS.</p>
</div>

<h2 id="switch" class="scia-h2 docs-subtitle">Switch (toggle)</h2>

The switch is made of:
- a hidden checkbox (<code class="scia-code">.scia-switch-input</code>)
- a visual track (<code class="scia-code">.scia-switch-track</code>)
- a thumb (<code class="scia-code">.scia-switch-thumb</code>)
- optional label text (<code class="scia-code">.scia-switch-label</code>)

<div className="playground-box">
```html
<!-- Notification Switch -->
<label className="scia-switch"> 
    <span className="scia-switch-label">Notifications</span> 
    <input className="scia-switch-input" type="checkbox" defaultChecked /> 
    <span className="scia-switch-track"> 
        <span className="scia-switch-thumb" /> 
    </span> 
</label> 

<!-- Disabled Switch -->
<label className="scia-switch"> 
    <span className="scia-switch-label">Disabled</span> 
    <input className="scia-switch-input" type="checkbox" disabled /> 
    <span className="scia-switch-track"> 
        <span className="scia-switch-thumb" /> 
    </span> 
</label>
```
<div className='playground-box-content'>
<div className='graphic-example-title'>Switch example</div>
<div style={{ display: "flex", flexDirection: "column", gap: "1.2rem" }}>
    <label className="scia-switch"> 
        <span className="scia-switch-label">Notifications</span> 
        <input className="scia-switch-input" type="checkbox" defaultChecked /> 
        <span className="scia-switch-track"> 
            <span className="scia-switch-thumb" /> 
        </span> 
    </label> 
    <label className="scia-switch"> 
        <span className="scia-switch-label">Disabled</span> 
        <input className="scia-switch-input" type="checkbox" disabled /> 
        <span className="scia-switch-track"> 
            <span className="scia-switch-thumb" /> 
        </span> 
    </label> 
</div>
</div>
</div>

<div class="scia-blockquote-info">
<p class="blockquote-paragraph">
- The checkbox is visually hidden but remains accessible.
- The track is the clickable surface and the thumb animates with <code class="scia-code">transform</code>.
</p>
</div>
