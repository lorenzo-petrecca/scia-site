<h1 id="intro" class="docs-title scia-typo-uppercase scia-typo-normal scia-typo-black">Utils</h1>

<p class="intro">
Scia CSS includes a set of small utilities for everyday layout and styling tasks:
colors, spacing, display/layout, sizing, radius/shadows, opacity, cursor, gap, positioning, and transitions.
</p>
<p class="intro">
Utilities are meant to be **composable**: you can stack multiple classes on the same element
to get predictable results without writing custom CSS.
</p>

<br/>

<h2 id="spacing" class="scia-h2 docs-subtitle">Spacing</h2>

All spacing utilities are based on <code class="scia-code">`--scia-space-*`</code> tokens.

<br/>


### Margin

Available directions:

- <code class="scia-code">`scia-margin-*`</code> (all)
- <code class="scia-code">`scia-margin-t-*`</code> (top)
- <code class="scia-code">`scia-margin-b-*`</code> (bottom)
- <code class="scia-code">`scia-margin-l-*`</code> (left)
- <code class="scia-code">`scia-margin-r-*`</code> (right)
- <code class="scia-code">`scia-margin-auto`</code> and directional <code class="scia-code">`*-auto`</code> helpers

<br/>
<br/>


### Padding

Available directions mirror margins:
- <code class="scia-code">`scia-padding-*`</code> (all)
- <code class="scia-code">`scia-padding-t-*`</code> (top)
- <code class="scia-code">`scia-padding-b-*`</code> (bottom)
- <code class="scia-code">`scia-padding-l-*`</code> (left)
- <code class="scia-code">`scia-padding-r-*`</code> (right)

<h2 id="display-layout" class="scia-h2 docs-subtitle">Display & layout</h2>

### Flex
Supported flex utilities include:

- <code class="scia-code">scia-flex</code>, <code class="scia-code">scia-inline-flex</code>
- <code class="scia-code">scia-flex-col</code>, <code class="scia-code">scia-flex-row</code>
- <code class="scia-code">scia-items-start|center|end</code>
- <code class="scia-code">scia-justify-center,</code> <code class="scia-code">scia-justify-between</code>
- <code class="scia-code">scia-flex-wrap</code>, <code class="scia-code">scia-flex-nowrap</code>

<br/>
<br/>

### Grid

<code class="scia-code">scia-grid-2</code> and <code class="scia-code">scia-grid-3</code> define equal columns with <code class="scia-code">minmax(0, 1fr)</code>.

<div className="playground-box">
```html
<div className="scia-grid scia-grid-3 scia-gap-12"> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">1</div> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">2</div> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">3</div> 
</div>
```
<div className='playground-box-content'>
<div className='graphic-example-title'>Grid example</div>
<div className="scia-grid scia-grid-3 scia-gap-12"> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">1</div> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">2</div> 
    <div className="scia-bg-on-surface scia-padding-12 scia-rounded-md">3</div> 
</div>
</div>
</div>

<br/>
<br/>

### Block / none

```html
<div class="scia-block">...</div>   <!-- display: block -->
<div class="scia-none">...</div>    <!-- display: none -->
```
