<h1 id="intro" class="docs-title scia-typo-uppercase scia-typo-normal scia-typo-black">About Scia CSS</h1>

Scia CSS is a lightweight, token-based CSS framework designed for clean defaults and fast UI building.
This website hosts the documentation and examples for the project.
